# Python業務自動化 学習プロジェクト

## プロジェクト概要

Pythonを活用した業務自動化の学習リポジトリ。
実務で使える自動化スクリプトの作成を通じて、Pythonの実践的なスキルを習得する。

## 学習目標

- Pythonによる業務自動化の基礎から応用まで習得
- 実務で即使えるスクリプトの作成
- コードの品質・可読性・保守性の向上

## 環境

- 言語: Python 3.x
- OS: Windows 11
- パッケージ管理: pip / venv

## ディレクトリ構成（予定）

```
python-automation/
├── CLAUDE.md
├── README.md
├── requirements.txt
└── scripts/          # 自動化スクリプト
```

## 開発ガイドライン

- 仮想環境 (`venv`) を使用する
- 依存パッケージは `requirements.txt` に記載する
- スクリプトには簡潔なコメントで目的を記載する
